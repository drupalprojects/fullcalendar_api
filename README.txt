INSTALLATION
------------

1. Download the latest FullCalendar release at http://fullcalendar.io/, unpack and place in /libraries.
3. Download Moment.js at https://github.com/moment/moment/releases, unpack and place in /libraries.
3. Enable FullCalendar API module.
4. Use the provided theme functions to integrate FullCalendar into your project.
